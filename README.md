# Mauritius Mass Finder — GitHub Pages v11

This bundle is prepared for simple static hosting on GitHub Pages.

## Files
- `index.html` — web app entrypoint
- `mauritius_mass_finder_final_v11_verified.csv` — verified dataset
- `Mauritius Mass Finder Ultimate Verification Report v11.txt` — verification note

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this bundle to the repository root.
3. In repository settings, open **Pages**.
4. Set source to **Deploy from a branch**.
5. Choose branch **main** and folder **/(root)**.
6. Save.
7. GitHub will generate a public URL.
